# SentinelStack

A production-grade fullstack system built from scratch as part of a structured engineering curriculum.

## Stack

- FastAPI
- Next.js
- PostgreSQL
- Bash automation
- Security / purple team practices

## Structure

- `backend/` - FastAPI backend
- `frontend/` - Next.js frontend
- `database/` - schema and migrations
- `scripts/` - setup, backup, deploy automation
- `security/` - audits and attack simulations
- `docs/` - logs, architecture, decisions

## Current Status

Environment setup in progress. Core scaffolding initialized.
